Use as setas do teclado para mover a tela
1 a 9 - Muda o sprite para o sprite de numero correspondente a barra superior
draw - Desenha o tile atual
delete - Deleta o tile atual
exit - Sair do editor
colliderMode - Ativa/Desativa o modo de colliders
nextTile - Muda para o proximo sprite
previousTile - Muda para o sprite anterior
nextToolbar - Proxima barra de tiles
previousToolbar - Barra de tiles anterior
info - Mostra/Esconde controles do teclado
grid - Mostra/Esconde a grade
favorite - Adiciona o sprite atual como favorito
highSpeed - Aumenta a velocidade da ferramenta
lowSpeed - Diminui a velocidade da ferramenta
shift+delete  - Copia tile atual
increaseColliderX - Aumentar tamanho X do collider
decreaseColliderX - Diminuir tamanho X do collider
increaseColliderY - Aumentar tamanho Y do collider
decreaseColliderY - Diminuir tamanho Y do collider
